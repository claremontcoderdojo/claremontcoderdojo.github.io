How to Use

  1. Download this file to your computer.
  2. Open this file in a plain text editor. (Like Atom or Notepad. Not in Microsoft Word.)
  3. Find the <script> tag at the bottom of this file. It is the one with the comment LOOK HERE!.
  4. Add your code between the <script> tags.
  5. Open the copy of this file on your in a web browser.
  6. Alter the code and hit refresh to see your changes.

It is helpful to save a different copy of this file for every project or subproject.
